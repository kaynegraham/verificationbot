# Verification Bot

Simple Discord verification bot built for FiveM communities.

This project provides basic user verification functionality and was created as a lightweight, easy-to-deploy solution using Node.js and Discord’s API.

The codebase is based on work by **notunderctrl**, with modifications and configuration tailored for FiveM community usage.

---

## Features

- Simple user verification workflow
- Lightweight and easy to configure
- Node.js based
- Designed for small to medium Discord communities

---

## Installation

1. Create a Discord application in the Discord Developer Portal  
2. Configure the bot via `config.json`  
3. Start the bot by running `run.bat` or using `node .` in a terminal

---

## Project Status

This project is considered complete and is not under active development.

It remains functional but may require updates if Discord or related APIs change in the future.
